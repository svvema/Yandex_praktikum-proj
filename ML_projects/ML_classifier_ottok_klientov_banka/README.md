# Отток клиентов из банка


## Данные

В наличии были следующие исторические данные от банка:
- CustomerId
- Surname
- CreditScore
- Geography
- Gender
- Age
- Tenure
- Balance
- NumOfProducts
- HasCrCard
- IsActiveMember
- EstimatedSalary
- Exited     

## Задача

Предсказать отток клиентов из банка по историческим данным.

## Используемые библиотеки
*pandas, plotly, matplotlib, numpy, sklearn*
